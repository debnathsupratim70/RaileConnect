# 🚆 RaileConnect

A modern full-stack railway ticket booking platform inspired by popular travel applications such as IRCTC, ixigo, and ConfirmTkt.

RaileConnect is being built as a portfolio-quality project using modern web technologies with a focus on clean architecture, responsive UI, and scalable backend design.

---

## ✨ Features

### Current Features

- Modern responsive homepage
- Train search form
- Search results page
- React Router navigation
- Reusable UI components
- TypeScript support
- Responsive layout
- GitHub version control

### Upcoming Features

- Station autocomplete
- Train availability
- Seat booking
- User authentication
- PNR status tracking
- Booking history
- Admin dashboard
- PostgreSQL database
- REST API
- JWT Authentication

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/debnathsupratim70/RaileConnect.git
```

### Go to Frontend

```bash
cd RaileConnect/frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
